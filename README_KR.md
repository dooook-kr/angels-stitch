# Angel's Stitch: Antigravity Pattern Maker

[English](README.md) | [Français](README_FR.md) | 한국어

## 📌 소개
십자수를 좋아하는 프랑스 어머니를 위해 만든 '십자수 도안 생성 및 온라인 작업 관리' 앱입니다. 원하는 사진을 쉽게 도안으로 만들어 다운로드 받을 수 있고, 종이 도안 없이 스마트폰이나 태블릿에서 스티치 진행 상황을 손쉽게 기록할 수 있습니다. 

Angel's Stitch는 프랑스 어머니의 성함인 Angelique에서 영감을 얻어 이름 지었습니다. 어머니의 소중한 취미 시간이 이 앱을 통해 더욱 즐겁고 평온한 시간이 되기를 바라는 마음을 담았습니다.

## ✨ 주요 기능
* **사진으로 도안 생성:** 이미지 업로드, 비율 크롭 및 색상 양자화 연산을 통해 사진을 십자수 도안으로 자동 변환합니다.
* **디지털 도안:** 웹에서 실시간으로 도안을 확인하고, 완성한 칸을 터치하여 작업 내용을 편리하게 트레킹할 수 있습니다.
* **고해상도 도안 PDF 다운로드:** 인쇄나 보관을 위해 선명한 화질의 PDF 도안 파일 다운로드를 지원합니다.
* **3단계 뷰 모드 전환:** 작업 편의를 위해 '분할 보기', '도안 전체 보기', '실 정보 전체 보기'의 세 가지 모드를 자유롭게 오갈 수 있습니다.
* **3개국어 지원:** 한국어, 영어, 프랑스어를 모두 지원하여 전 세계 어디서든 편하게 사용 가능합니다.
* **모바일 최적화 및 제스처 제어:** 모바일 기기에 최적화된 화면과 손가락 제스처(확대/축소, 이동)로 도안을 직관적으로 조작합니다.

## 🚀 향후 계획
더 나은 사용 경험을 위해 다음과 같은 업데이트를 준비하고 있습니다.
* **스티치 모드 최적화:** 터치 반응 속도를 개선하고, 도안을 더 편하게 볼 수 있도록 가독성을 높일 예정입니다.
* **밝은 테마(Bright Mode) 지원:** 눈의 피로를 줄이고 취향에 맞게 선택할 수 있는 밝은 배경 모드를 추가할 계획입니다.
