# Requirements Document: Antigravity Pattern Maker

## 1. 개요
사용자가 업로드한 이미지를 분석해 자수 실 데이터와 매칭하고 십자수 도안으로 변환하는 모바일 웹 기반 애플리케이션이다. 기존 프로토타입의 UI 컨셉을 유지하면서, 디지털 환경에 최적화된 스티치 트래킹과 종합 PDF 출력 기능을 통해 작업 생산성을 극대화한다.

## 2. 시스템 개선 요구사항

### 2.1. 유저 사이즈 (디바이스 대응 및 화면 비율)
- 뷰포트 및 스케일링 정책 변경: 고정된 뷰포트 설정을 해제하고, 캔버스 뷰어 내 독립적인 터치 제스처를 지원하도록 설계한다.
- 반응형 레이아웃 도입: 다양한 기기 해상도에 맞춰 UI가 유동적으로 변하는 그리드 시스템을 구축한다.

### 2.2. 디자인 및 인터랙션 (사용성)
- 제스처 기반 캔버스 제어: 핀치 투 줌 및 드래그 팬 액션을 통해 도안 탐색을 직관적으로 개선한다.
- 설정 피드백 루프 개선: 그리드 및 색상 변경 시 저해상도 썸네일 기반의 빠른 프리뷰를 제공하여 탐색 시간을 단축한다.

### 2.3. 개발 측면 (성능 및 데이터 안정성)
- Web Worker 기반 비동기 처리: 고부하 연산을 메인 스레드에서 분리하여 앱 프리징 현상을 차단한다.
- IndexedDB 스토리지 마이그레이션: 대용량 픽셀 맵과 작업 진행 데이터를 안전하게 영구 보존한다.

## 3. 핵심 기능 명세

### 3.1. 홈 및 프로젝트 관리
- IndexedDB 연동을 통한 기존 프로젝트 로드 및 삭제 기능.

### 3.2. 작업 공간
- 사진첩 연동 이미지 업로드, 비율 크롭, 비동기 이미지 양자화 연산.

### 3.3. 디지털 스티치 트래킹
- 온라인 작업 모드: 출력물 없이 웹 브라우저상에서 실시간으로 도안을 보며 작업할 수 있는 환경 제공.
- 스티치 마킹: 완성한 칸을 터치하여 완료 상태로 표시하고, 이를 IndexedDB에 실시간 저장하여 재접속 시 데이터 유지.
- 진척도 시각화: 전체 스티치 대비 완료된 스티치 수를 계산하여 진행률 퍼센티지 표시.

### 3.4. 종합 PDF 출력 엔진
- 단순 이미지 저장을 대체하는 고해상도 PDF 리포트 생성.
- 커버 페이지: 프로젝트 이름, 완성 예상 날짜, 완성 미리보기 이미지.
- 도안 페이지: 기호 및 컬러가 포함된 고해상도 그리드 도안.
- 구매 정보: 사용된 DMC 실 코드 리스트, 필요 스티치 수, 예상 타래 수.
